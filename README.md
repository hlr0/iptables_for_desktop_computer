 Iptables Script for Desktop Computer Setup

Usage: ./iptables.sh [OPTIONS]

Options:
  -h, --help     Show this help message and exit
  -c, --clean    Revert all changes and flush the iptables rules,
                 chains, and tables allowing to start from a clean
                 slate or to revert any changes made to the system

Examples:
  ./iptables.sh             Run the script with default firewall rules
  ./iptables.sh --clean     Flush all iptables rules and set default policies to ACCEPT
  ./iptables.sh --help      Display this help message

Note: This script must be run with root privileges
